# 使用教學

1. 前往[庫街區合鳴效果目錄頁](https://wiki.kurobbs.com/mc/catalogue/list?fid=1099&sid=1219)
2. 使用 Ctrl + S 保存合鳴效果目錄頁
3. 進入[任意合鳴套裝頁面](https://wiki.kurobbs.com/mc/catalogue/list?fid=1099&sid=1219)
4. 使用 Ctrl + S 保存合鳴套裝頁面
5. 前往[庫街區共鳴者目錄頁](https://wiki.kurobbs.com/mc/catalogue/list?fid=1099&sid=1105)
6. 使用 Ctrl + S 保存共鳴者目錄頁
7. 將頁面移動到此目錄下
8. 執行 parser.py 腳本
